# Uber Print Roadmap (Full Interactive Version)
هذه هي النسخة الكاملة من لوحة التنفيذ التفاعلية لمشروع "أوبر للطباعة" ✅

## طريقة التشغيل
1. افتح الملف `index.html` في متصفح حديث (Chrome أو Edge أو Firefox).
2. تقدّمك يُحفظ تلقائيًا في Local Storage.
3. يمكنك استخدام أزرار التصدير والاستيراد داخل الموقع.

## طريقة الرفع على GitHub Pages
1. أنشئ مستودع جديد باسم `uber-print-roadmap` على GitHub.
2. ارفع هذا الملف `index.html` إلى المستودع.
3. من الإعدادات → Pages → غيّر Source إلى الفرع `main` والمجلد `/ (root)`.
4. سيظهر لك الرابط المباشر للموقع بعد دقيقة تقريبًا.
